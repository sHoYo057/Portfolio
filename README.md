A convolutional neural network (CNN) that achieves **98%+ accuracy** on MNIST handwritten digit classification.

![Demo](https://github.com/sHoYo057/DIGITRECO/blob/main/images/demo.gif?raw=true)

## 🚀 Features

- 📸 Image preprocessing pipeline
- 🧠 CNN architecture with 2 convolutional layers
- ⚡ GPU acceleration support
- 📊 Real-time prediction visualization
- 🔢 Supports digits 0-9
